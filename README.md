## Key Features

* **Secure Authentication** - User registration and login with Firebase.
* **Real-Time Communication** - Instant chat and messaging system.
* **Project Management** - Tools for task tracking and deadline management.
* **Resource Sharing** - Centralized document storage and sharing.
* **Event Management** - Schedule and manage important meetings and deadlines.
* **Customizable Dashboard** - Personalized user dashboards.
* **Notifications** - Real-time updates on important changes.
* **Analytics and Reporting** - Insights into team performance and project progress.
* **Cross-Platform** - Compatible with Windows, macOS, and Linux.

## How To Use

To clone and run this application, you'll need [Git](https://git-scm.com) and [Node.js](https://nodejs.org/en/download/) (which includes [npm](http://npmjs.com)) installed on your computer. From your command line:

### Frontend

1. Navigate to the `frontend` directory:

    ```bash
    cd frontend
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Start the frontend:

    ```bash
    node app.js
    ```

### Backend

1. Navigate to the `backend` directory:

    ```bash
    cd backend
    ```

2. Install dependencies:

    ```bash
    npm i
    ```

3. Start the backend:

    ```bash
    npm start
    ```

### Game

1. Navigate to the `game` directory:

    ```bash
    cd game
    ```

2. Install dependencies and start the client game:

    ```bash
    yarn && yarn start
    ```

3. Navigate to the `game/client` directory:

    ```bash
    cd client
    ```

4. Install dependencies and start the client game:

    ```bash
    yarn && yarn dev
    ```
